# 2 Sigma Learning

[English README](README.md)

用本地 AI 老师学任何东西。它一小块一小块地教，随时提问，卡住就换一种讲法；就算换一个全新会话，也能只靠本地文件接着教。

2 Sigma Learning 是一个 Claude Code skill，适合那些不想只拿到一个答案、而是想真的学会一门东西的人。你只能用 `/2-sigma-learning` 启动它。启动后，它会在当前文件夹里跑一套持续课程：先定计划，再分块讲解，当场提问，记录学习证据，再决定下一节怎么接着教。

## 这东西为什么存在 (Why this exists)

1980 年代，Benjamin Bloom 提出过一个后来很有名的结论，叫 **2 Sigma problem**。意思很直接：如果满足两个条件，一个学生的表现可以超过同龄人里 98% 的人。

这两个条件不神秘，但以前很贵：

1. **一对一辅导 (One-to-one tutoring)**  
   一个老师只带一个学生，节奏跟着理解程度走。
2. **掌握学习法 (Mastery learning)**  
   不是一章讲完就翻篇，而是当前内容真学到位了，再往下走。通常要到 80% 到 90% 的掌握度。

问题不在理论，问题在成本。长期家教太贵，所以这套最有效的学习条件，过去更像少数人的特权。听不懂的人，在大班里只能硬跟。

AI 第一次把这个循环的成本压下来了。本地 Agent 现在可以长期陪你学，还能把学习痕迹留在你的电脑里。

但普通 AI 对话还有个根本问题：你越不懂一个领域，越不知道该问什么。你得到的是一堆宽泛答案，不是学习过程。

真正有用的，是一个会反过来问你的老师：

- 你到底想学到什么程度？
- 你现在已经会什么？
- 这个概念你能不能讲回来？
- 你该继续，还是该换一种讲法？

这就是这个 skill 要做的事。

## 这玩意能干啥 (What this does)

`2-sigma-learning` 不是拿来“问一个问题然后散场”的。它是拿来在本地跑一门持续课程的。

它会：

- 在空文件夹里开始一门新课
- 读取 `proposal.md` 和已有课时文件，继续原来的课程
- 不靠聊天记忆，而是靠本地文件恢复教学状态
- 每次只教一个 chunk，不一次性倒一整课
- 每个 chunk 后面立刻提 1–2 个检查问题
- 先按弱点做 warm-up review，再进入新内容
- 按大约 80% 到 90% 掌握度决定能不能往下走
- 你卡住时换一种解释路径，不重复失败表达
- 创建 `Class 02b - <topic> (remediation).md` 这种补救课
- 把学习记录写回本地文件
- 课程结束时生成 `summary.md`

## 咋开始，先跑起来 (How to start)

这个 skill 只用斜杠命令触发：

```text
/2-sigma-learning
```

触发之后，你再告诉它你想学什么，或者说你要继续当前文件夹里的已有课程。

如果当前目录已经有 `proposal.md`，它会默认走继续课程。如果目录是空的，它会默认新开一门课。如果目录里有一堆无关文件，它会建议你建一个 `./courses/<slug>/` 子文件夹。

## 它会问你啥 (What it asks from you)

新课程开始时，它只问够用的问题：

- 你想学到什么地步
- 你现在会什么，弱点在哪
- 你打算花多少时间
- 你想学到什么深度
- 用通用知识就行，还是要吃你给的教材、论文、文档

然后它会先写出 `proposal.md`，让你看课程计划对不对，再一起改。

真正上课的时候，它不会一口气讲个没完。它会讲一个 chunk，接着出 1–2 个检查题，看你答得怎么样，再决定是继续、复习、来一道更难的题，还是换一条讲法。

## 它会写出啥 (What it writes)

一门课默认长这样：

```text
proposal.md
Class 01 - <topic>.md
Class 02 - <topic>.md
Class 02b - <topic> (remediation).md
...
summary.md
```

`proposal.md` 是课程级索引。它放课程目标、知识地图、课程序列、当前进度、需要反复热身的弱概念，还有更新日志。

每个 class 文件是这一课的证据文件。它不只是教材，也记录未来某个新会话继续上课时需要知道的东西。

class 文件里的结构标题固定用英文，即使正文是中文。这样做是为了让不同语言的课程都能被稳定读取。常见关键部分包括：

- `## At a Glance`
- `## Key Takeaways`
- `## Textbook`
- `## Misconceptions / Weak Concepts`
- `## Learning Record`
- `## Mastery & Gaps`
- `## Hooks for Next Class`

每节课结束时，它必须做四件事：

1. 写入或更新 class 文件，把本节真实学习证据放进去
2. 更新 `proposal.md`，记录进度和要带到后面的弱概念
3. 搭好下一节课，或者搭一个补救课文件
4. 问你是现在继续，还是停在这里下次再来

它不能编造教材正文、例题、你的回答或学习记录。如果下一节还没准备好，它只会先搭一个骨架，不会硬编一整课。

## 为啥一定要落到本地文件 (Why local files matter)

学习这件事，最怕没记忆。普通对话框一关，很多关键东西就断了。

V2 的核心约束是：就算开一个完全没有聊天记忆的新会话，也必须能只靠本地文件继续教学。

所以文件里必须留下：

- 你已经懂了什么
- 你老是在哪误解
- 哪条解释路径根本没用
- 哪个解释终于讲通了
- 哪些弱点下节课得先热身
- 下一节课应该从哪里接上

这就是“普通 AI 帮忙”和“真的在带你上课”之间的差别。

## 卡住时它怎么换讲法 (How it adapts when you are stuck)

它不会把同一句话换个说法再来一遍。

如果一个核心概念低于 60% 掌握度，它必须换一条解释路径：

1. 抽象定义或形式化说法
2. 具体例子
3. 类比
4. 反例或对比
5. 图示、表格或结构化解释
6. 回到更基础的前置概念

它还会记录哪条路讲通了。下一次继续上课时，就能复用有效讲法，避开没用的讲法。

## 适合拿来干这些 (Use it for)

这玩意适合你在下面这些场景用：

- 进一个完全陌生的领域，不想对着空白输入框发呆
- 想把一个大主题拆成一节一节来学
- 想拿自己的教材、论文、文档来喂课程
- 想跨会话保留学习进度
- 想逼自己做 retrieval practice，而不是光看不答
- 想先修掉薄弱概念，再进入下一课

## 别指望它包打天下 (Limits)

这是一个学习流程，不是资格认证。

- 它不能替代正式教学、专业监督或领域专家。
- 医疗、法律、金融、安全这类高风险领域，关键内容要自己查权威来源。
- 掌握度判断只是教学判断，不是正式考试成绩。
- 只有你明确输入 `/2-sigma-learning`，它才会运行。
