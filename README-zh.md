# 2 Sigma Learning

[English README](README.md)

用本地 AI 老师学任何东西。它一小块一小块地教，随时提问，卡住就换一种讲法，还会把你的进度和卡点记在文件里。

2 Sigma Learning 是一个 Claude Code skill，适合那些不想只拿到一个答案、而是想真的学会一门东西的人。你用 `/2-sigma-learning` 启动它，它就会在当前文件夹里跑一套持续课程：先定计划，再分块讲解，当场提问，记录学习证据，再决定下一节怎么接着教。

## 这东西为什么存在 (Why this exists)

1980 年代，Benjamin Bloom 提出过一个后来很有名的结论，叫 **2 Sigma problem**。意思很直接：如果满足两个条件，一个学生的表现可以超过同龄人里 98% 的人。

这两个条件不神秘，但以前很贵：

1. **一对一辅导 (One-to-one tutoring)**  
   一个老师只带一个学生，节奏跟着理解程度走。
2. **掌握学习法 (Mastery learning)**  
   不是一章讲完就翻篇，而是当前内容真学到位了，再往下走。通常要到 80% 到 90% 的掌握度。

问题不在理论，问题在成本。长期家教太贵，所以这套最有效的学习条件，过去更像少数人的特权。听不懂的人，在大班里只能硬跟。

AI 第一次把这个循环的成本压下来了。本地 Agent 现在可以长期陪你学，还能把学习痕迹留在你的电脑里。

但普通 AI 对话还有个根本问题：你越不懂一个领域，越不知道该问什么。你得到的是一堆宽泛答案，不是学习过程。

真正有用的，是一个会反过来问你的老师：

- 你到底想学到什么程度？
- 你现在已经会什么？
- 这个概念你能不能讲回来？
- 你该继续，还是该换一种讲法？

这就是这个 skill 要做的事。

## 这玩意能干啥 (What this does)

`2-sigma-learning` 不是拿来“问一个问题然后散场”的。它是拿来在本地跑一门持续课程的。

它会：

- 在空文件夹里开始一门新课
- 读取 `proposal.md` 和已有课时文件，继续原来的课程
- 每次只教一个 chunk，不一次性倒一整课
- 每个 chunk 后面立刻提 1–2 个检查问题
- 先按弱点做 warm-up review，再进入新内容
- 按大约 80% 到 90% 掌握度决定能不能往下走
- 你卡住时换一种解释路径，不重复失败表达
- 把学习记录写回本地文件
- 课程结束时生成 `summary.md`

## 咋开始，先跑起来 (How to start)

这个 skill 只用斜杠命令触发：

```text
/2-sigma-learning
```

触发之后，你再告诉它你想学什么，或者说你要继续当前文件夹里的已有课程。

如果当前目录已经有 `proposal.md`，它会默认走继续课程。如果目录是空的，它会默认新开一门课。如果目录里有一堆无关文件，它会建议你建一个 `./courses/<slug>/` 子文件夹。

## 它会问你啥 (What it asks from you)

新课程开始时，它只问够用的问题：

- 你想学到什么地步
- 你现在会什么，弱点在哪
- 你打算花多少时间
- 你想学到什么深度
- 用通用知识就行，还是要吃你给的教材、论文、文档

然后它会先写出 `proposal.md`，让你看课程计划对不对，再一起改。

真正上课的时候，它不会一口气讲个没完。它会讲一个 chunk，接着出 1–2 个检查题，看你答得怎么样，再决定是继续、复习，还是换一条讲法。

## 它会写出啥 (What it writes)

一门课默认长这样：

```text
proposal.md
Class 01 - <topic>.md
Class 02 - <topic>.md
...
summary.md
```

`proposal.md` 里会放课程目标、课程序列、当前进度，还有那些没学扎实、后面热身还要再拉出来问的概念。

每个 class 文件里会放：

- 本课目标
- 热身复习
- 分 chunk 的教材
- 检查问题
- 练习题
- 这一课的评估标准
- `## Learning Record`

每节课结束后，它会把这些写回去：

- 这节讲了什么
- 问了哪些题
- 你是怎么答的
- 它给了什么反馈
- 每个点大概掌握到什么程度
- 还差在哪
- 哪个类比或解释对你最管用
- 下一节该怎么接

课程结束时，它会生成 `summary.md`，把你已经拿下的概念、还虚的地方、以及后续建议整理出来。

## 为啥一定要落到本地文件 (Why local files matter)

学习这件事，最怕没记忆。普通对话框一关，很多关键东西就断了。

本地 Agent 可以一直记着：

- 你已经懂了什么
- 你老是在哪误解
- 哪种解释根本没用
- 哪个解释终于讲通了
- 哪些弱点下节课得先热身

这就是“普通 AI 帮忙”和“真的在带你上课”之间的差别。

## 适合拿来干这些 (Use it for)

这玩意适合你在下面这些场景用：

- 进一个完全陌生的领域，不想对着空白输入框发呆
- 想把一个大主题拆成一节一节来学
- 想拿自己的教材、论文、文档来喂课程
- 想跨会话保留学习进度
- 想逼自己做 retrieval practice，而不是光看不答

## 别指望它包打天下 (Limits)

这是一个学习流程，不是资格认证。

- 它不能替代正式教学、专业监督或领域专家。
- 医疗、法律、金融、安全这类高风险领域，关键内容要自己查权威来源。
- 掌握度判断只是教学判断，不是正式考试成绩。
- 只有你明确输入 `/2-sigma-learning`，它才会运行。
